# Changelog
All notable changes to this project are documented in this file.


## [0.1.0] - 2026-01-16

### Added
- Initial project structure and generator package
- `NumberGenerator` interface
- `AbstractGenerator` base class
- `SortedGenerator` for fully sorted input lists
- `SwapSortGenerator` for partially sorted inputs using a rate number of random swaps
- Project planning file (`TODOs.md`)
